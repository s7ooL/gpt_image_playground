## v0.6.9（2026-06-19）

### 新增
- **支持为 Agent 模式使用独立的 API 配置**：支持以下两种模式：
  - **原生**：使用原生的 Responses API 配置，由模型调用 `image_generation` 工具生成图片。
  - **混合**：使用非原生的混合 API 配置，由文本模型调用自定义工具，请求图像模型生成图像，解决部分服务商/模型不支持 `image_generation` 工具的问题。
- **支持允许模型改写优化提示词** (Issue #106)：常规设置中新增“允许模型改写优化提示词”开关。开启后，在 Codex CLI 兼容模式下的 Image API 请求和所有 Responses API 请求中不再强制附加防改写提示词，模型将能够自由优化提示词。

### 变更与优化
- **超大组件模块化重构**：为提升代码可维护性，将收藏夹、输入栏和设置弹窗等超大组件进行了拆分。
- **优化 ZIP 图片导出命名**：重构并封装导出逻辑，导出的图片改为以生成任务 ID（如 `task-[id]`) 结合对应角色（如 `-input` / `-partial` / `-mask` 等）进行智能重命名，提升了文件名可读性。
- **调整请求中间步骤图像数的选择菜单样式**：将请求中间步骤图像数的选择菜单与选项名称调整到同一行显示，使其更加紧凑美观。

### 修复
- **修复 Tooltip 遮挡溢出**：在 Tooltip 悬浮提示中增加视口遮挡命中检测，当触发元素被上层弹窗或遮罩层遮挡时自动隐藏，防止悬浮提示穿透溢出到弹窗上层。
- **修复收藏夹弹窗样式一致性**：将收藏夹选择及管理弹窗的样式对齐应用整体的视觉风格。
